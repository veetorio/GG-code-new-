    Inicio de tudo

    • 1991 : James Gosling, Patrick Naughton, Chris Warth, Ed Frank e Mike Sheridan começam a trabalhar no projeto Green na Sun Microsystems, com o objetivo de desenvolver software para dispositivos portáteis. Eles inicialmente chamam a linguagem de Oak, inspirados por uma árvore que estava fora da janela de Gosling. Mais tarde, o nome é alterado para Java.

    • 1995: Java 1.0 é lançado pela Sun Microsystems em 23 de maio. O lançamento inclui o compilador Java, o Java Virtual Machine (JVM) e o conjunto inicial de bibliotecas Java (APIs). A plataforma Java é promovida como "write once, run anywhere" (escreva uma vez, execute em qualquer lugar), permitindo que o código Java seja executado em qualquer dispositivo que tenha uma JVM. |

    • 1996: A Microsoft lança sua própria versão do Java, que inclui extensões proprietárias, levando a uma disputa legal entre a Sun e a Microsoft sobre a compatibilidade da implementação.

    O que é o java ?

    Java é uma linguagem de programação multiplataforma baseada em orientação a objetos. Com sua capacidade de compilar para bytecode, permite que os programas sejam executados em diferentes sistemas operacionais. Sua natureza orientada a objetos promove conceitos como encapsulamento, herança e polimorfismo, tornando-a uma escolha popular para desenvolvimento de uma variedade de aplicações.

    O que são JDK,JRE e JVM?

    JDK

    é um conjunto de ferramentas de software usado por desenvolvedores Java para criar aplicativos, applets e componentes Java. Ele contém tudo o que é necessário para desenvolver, compilar, depurar e executar aplicativos Java.

    JRE

    é um componente essencial do ambiente Java que permite a execução de aplicativos Java em uma máquina virtual Java (JVM).

    JVM

    é uma máquina virtual que interpreta e executa o bytecode Java. Ela é um componente essencial do ambiente de execução Java e é responsável por traduzir o código bytecode Java em instruções compreensíveis pelo hardware e sistema operacional subjacentes.
